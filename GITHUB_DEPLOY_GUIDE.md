# 📱 बिना कंप्यूटर: मोबाइल से सीधे GitHub पर अपलोड और लाइव करने की पूरी गाइड

यदि आपके पास कंप्यूटर नहीं है, तो भी आप **100% मोबाइल का उपयोग करके** इस स्कूल ERP ऐप को अपनी खुद की GitHub रिपोजिटरी में डाल सकते हैं और इसे `https://<your-username>.github.io/school-erp/` लिंक पर लाइव चला सकते हैं!

हमने इसके लिए एक **स्वचालित क्लाउड बिल्ड सिस्टम (GitHub Actions)** सेट कर दिया है। इसका मतलब है कि आपको मोबाइल पर कोई कोड कंपाइल करने की ज़रूरत नहीं है—जैसे ही आप फाइल्स को GitHub पर डालेंगे, GitHub का सर्वर खुद ही बैकग्राउंड में ऐप को कंपाइल करके लाइव कर देगा!

---

## 🛠️ मोबाइल से सेटअप करने के 2 सबसे आसान तरीके (Choose any 1)

### तरीका 1: "Spck Editor" ऐप का उपयोग करके (सबसे आसान और बेस्ट तरीका 🌟)
Spck Editor मोबाइल के लिए एक मुफ़्त और बहुत ही बेहतरीन कोड एडिटर और Git क्लाइंट है। इससे आप 1 क्लिक में कोड GitHub पर भेज सकते हैं।

1. **ऐप डाउनलोड करें:** प्ले स्टोर (Play Store) या ऐप स्टोर (App Store) से **Spck Editor** इंस्टॉल करें।
2. **कोड डाउनलोड करें:** AI Studio में ऊपर दाईं ओर **Settings (गियर आइकन)** पर क्लिक करके **Export as ZIP** पर क्लिक करें और कोड को अपने मोबाइल में डाउनलोड कर लें।
3. **Spck Editor में इम्पोर्ट करें:**
   - ऐप खोलें और लेफ्ट साइड मेनू में जाएं।
   - **Import ZIP** विकल्प चुनें और अपने डाउनलोड किए गए प्रोजेक्ट की ZIP फाइल को सिलेक्ट करें।
   - आपका पूरा प्रोजेक्ट अब आपके मोबाइल एडिटर में आ चुका है!
4. **GitHub अकाउंट लिंक करें:**
   - Spck Editor के साइड मेनू में **Git** पर जाएं।
   - **Accounts** पर क्लिक करें और अपने GitHub अकाउंट को लॉग-इन/लिंक करें (इसके लिए आप GitHub का Access Token उपयोग कर सकते हैं)।
5. **नया प्रोजेक्ट GitHub पर भेजें:**
   - Spck Editor में प्रोजेक्ट के नाम पर लॉन्ग-प्रेस करें और **Git -> Remote** पर जाएं।
   - अपनी नई GitHub रिपोजिटरी का URL (जो आपने GitHub पर बनाया है, जैसे: `https://github.com/username/school-erp.git`) वहां डालें।
   - अब **Git -> Push** पर क्लिक करें। आपका पूरा कोड सीधे आपके मोबाइल से GitHub पर अपलोड हो जाएगा!

### तरीका 2: मोबाइल ब्राउज़र (Desktop Site) का उपयोग करके (बिना किसी ऐप के)
यदि आप कोई ऐप डाउनलोड नहीं करना चाहते, तो आप सीधे Chrome/Safari ब्राउज़र से भी कर सकते हैं:

1. **ZIP फ़ाइल डाउनलोड करें:** AI Studio से प्रोजेक्ट को **ZIP** के रूप में डाउनलोड करें और अपने मोबाइल के File Manager में उसे **Unzip (Extract)** कर लें।
2. **GitHub पर जाएं:** मोबाइल ब्राउज़र में [github.com](https://github.com/) खोलें और अपने अकाउंट में लॉगिन करें।
3. **डेस्कटॉप मोड ऑन करें:** ब्राउज़र की सेटिंग्स में जाकर **"Desktop Site" (डेस्कटॉप साइट)** विकल्प चालू करें।
4. **नई रिपोजिटरी बनाएं:**
   - **New Repository** बटन पर क्लिक करें।
   - नाम दें: `school-erp` और इसे **Public** रखें।
5. **फाइल्स अपलोड करें:**
   - रिपोजिटरी बनने के बाद आपको **"uploading an existing file"** का एक छोटा सा लिंक दिखेगा, उस पर क्लिक करें।
   - अपनी अनज़िप की हुई फाइल्स और फोल्डर्स (जैसे `index.html`, `package.json`, `src/`, `.github/` आदि) को सिलेक्ट करके अपलोड कर दें।
   - नीचे **Commit changes** पर क्लिक करें।

## 🚀 लाइव लिंक कैसे सक्रिय होगा? (How to make the Link Live)

हमने आपकी फाइल्स में `.github/workflows/deploy.yml` नाम की एक फ़ाइल जोड़ी है। जैसे ही आपका कोड GitHub पर पहुंचेगा:

1. GitHub का **Actions** सिस्टम खुद ही चालू हो जाएगा और बैकग्राउंड में आपके प्रोजेक्ट को कंपाइल करना शुरू कर देगा (इसमें करीब 1-2 मिनट लगते हैं)।
2. जब कम्पाइलेशन पूरा हो जाएगा, तो आपके GitHub अकाउंट में खुद-ब-खुद **`gh-pages`** नाम की एक ब्रांच बन जाएगी।
3. **लाइव लिंक देखने के लिए:**
   - अपनी GitHub रिपोजिटरी के **Settings** टैब में जाएं।
   - बाईं ओर मेनू में **Pages** पर क्लिक करें।
   - **Build and deployment** के नीचे:
     - Source: **Deploy from a branch** रखें।
     - Branch: **`gh-pages`** सिलेक्ट करें और **`/ (root)`** रखें।
     - **Save** पर क्लिक करें।
4. अब पेज को रीलोड करें। आपको ऊपर आपका लाइव लिंक दिखाई देगा:
   👉 **`https://<आपका-GitHub-यूज़रनेम>.github.io/school-erp/`**

इसे आप सीधे अपने मोबाइल में खोल सकते हैं, सेव कर सकते हैं, या किसी के साथ भी शेयर कर सकते हैं!
